# Tower of Hanoi

세 기둥 사이에서 원반을 옮겨 모든 원반을 오른쪽 기둥으로 이동시키는 고전 논리 퍼즐입니다.

## 실행

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 주소는 `http://127.0.0.1:5191`입니다.

## 조작

- 마우스 클릭: 기둥 선택 및 이동
- 좌우 방향키: 선택 기둥 이동
- Space 또는 Enter: 현재 기둥 선택
- `N`: 새 게임
- `P`: 일시정지 또는 재개

## 규칙

- 한 번에 맨 위 원반 하나만 옮길 수 있습니다.
- 큰 원반은 작은 원반 위에 놓을 수 없습니다.
- 모든 원반을 오른쪽 기둥으로 옮기면 클리어됩니다.
- Easy, Normal, Hard 난이도는 원반 개수와 점수 기준이 다릅니다.

## 점수

- 난이도별 기본 점수에서 경과 시간과 최소 이동 수 초과분을 차감합니다.
- 이론상 최소 이동 수는 `2^원반수 - 1`입니다.
- 개인 최고 점수는 브라우저 `localStorage`에만 저장됩니다.

## 허브 연동

- 게임 시작 시 `minigame:start` 메시지를 부모 창에 전송합니다.
- 점수 변경 시 `minigame:score` 메시지를 전송합니다.
- 클리어 또는 허브 종료 명령 시 `minigame:game-over` 메시지를 전송합니다.
- 허브의 `start`, `pause`, `gameOver` 라이프사이클 명령을 처리합니다.
