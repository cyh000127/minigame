# Sudoku

9x9 격자에 1부터 9까지의 숫자를 채워 각 행, 열, 3x3 박스에 같은 숫자가 중복되지 않게 만드는 퍼즐 게임입니다.

## 실행

```bash
pnpm --dir games/sudoku install
pnpm --dir games/sudoku dev
```

기본 주소는 `http://127.0.0.1:5188`입니다.

## 규칙

- Easy, Normal, Hard 난이도를 선택할 수 있습니다.
- 난이도가 높을수록 처음에 주어지는 숫자가 줄어듭니다.
- 퍼즐은 solver로 유일해가 확인된 상태만 생성합니다.
- 정답과 다른 숫자를 입력하면 실수 횟수가 증가하고 칸이 강조됩니다.
- 힌트는 선택한 칸 또는 아직 맞지 않은 칸 하나를 정답으로 채웁니다.

## 조작

- 마우스/터치: 칸 선택, 숫자 패드 입력
- 숫자키 1-9: 선택한 칸에 숫자 입력
- Backspace/Delete/0: 선택한 칸 지우기
- 방향키: 선택 칸 이동
- H: 힌트 사용
- N: 새 게임

## 점수

클리어 시 난이도별 기본 점수에서 시간, 실수, 힌트 페널티를 차감합니다. 클리어 점수는 허브에 `minigame:game-over` 메시지로 전달됩니다.
