# Lights Out

켜진 불을 모두 끄는 격자 퍼즐 게임입니다. 칸을 누르면 자기 칸과 상하좌우 칸이 함께 토글됩니다.

## 실행

```bash
pnpm --dir games/lights-out install
pnpm --dir games/lights-out dev
```

기본 주소는 `http://127.0.0.1:5190`입니다.

## 규칙

- Easy는 3x3, Normal은 5x5, Hard는 7x7 보드입니다.
- 칸을 누르면 해당 칸과 상하좌우 칸의 켜짐/꺼짐이 바뀝니다.
- 각 라운드에는 넉넉한 제한시간이 있습니다.
- 모든 불을 끄면 점수를 얻고 새 맵으로 즉시 리셋됩니다.
- 제한시간이 끝나면 실패로 처리되고 그 시점의 누적 점수가 기록됩니다.
- 퍼즐은 꺼진 보드에서 실제 토글을 적용해 생성하므로 항상 풀 수 있습니다.

## 조작

- 마우스/터치: 칸 토글
- 방향키/WASD: 선택 칸 이동
- Space/Enter: 선택 칸 토글
- N: 새 게임

## 점수

라운드 클리어 시 난이도, 라운드, 남은 시간에 따라 점수를 얻고 이동 횟수만큼 보너스가 줄어듭니다. 시간초과 또는 허브 종료 시 누적 점수를 `minigame:game-over` 메시지로 전달합니다.
