# Math Rush

화면 중앙의 수식을 보고 네 방향키에 배치된 정답 후보 중 맞는 값을 빠르게 고르는 산술 반응 게임입니다.

## 실행

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 주소는 `http://127.0.0.1:5193`입니다.

## 조작

- 방향키: 해당 방향의 답 선택
- Space 또는 Enter: 대기 상태에서 시작
- `N`: 새 게임
- `P`: 일시정지 또는 재개
- 마우스 클릭: 답 버튼 선택

## 규칙

- 제한 시간 안에 정답을 고르면 다음 라운드로 넘어갑니다.
- 라운드가 오를수록 제한 시간이 점점 줄어듭니다.
- 오답은 남은 시간을 깎고 연속 정답을 초기화합니다.
- 5연속 정답부터 피버가 켜지고 정답 점수가 2배가 됩니다.

## 점수

- 난이도별 기본 점수, 라운드, 남은 시간, 연속 정답 수를 기준으로 점수를 얻습니다.
- Easy, Normal, Hard는 시작 제한 시간과 감속 폭, 오답 패널티가 다릅니다.
- 개인 최고 점수는 브라우저 `localStorage`에만 저장됩니다.

## 허브 연동

- 게임 시작 시 `minigame:start` 메시지를 부모 창에 전송합니다.
- 점수 변경 시 `minigame:score` 메시지를 전송합니다.
- 제한 시간이 끝나거나 허브 종료 명령을 받으면 `minigame:game-over` 메시지를 전송합니다.
- 허브의 `start`, `pause`, `gameOver` 라이프사이클 명령을 처리합니다.
