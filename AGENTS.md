# 작업 지침

이 저장소는 미니게임을 하나씩 독립 디렉토리에 구현하는 저장소입니다.

## 코드 접근 범위

- 새 게임을 만들거나 기존 게임을 수정할 때는 저장소 루트 문서와 대상 게임 디렉토리만 읽습니다.
- 다른 게임 디렉토리의 코드는 참고, 복사, import하지 않습니다.
- 전체 검색이 필요할 때도 `docs/`, `README.md`, `AGENTS.md`, 대상 게임 디렉토리로 범위를 제한합니다.
- 공통 코드가 필요해 보여도 공유 디렉토리를 만들지 않고, 해당 게임 안에 필요한 코드를 둡니다.

## 게임 디렉토리 규칙

- 새 게임은 `games/<game-name>/`에 만듭니다.
- 게임 디렉토리에는 해당 게임의 실행 방법을 설명하는 `README.md`를 둡니다.
- 게임 코드는 TypeScript로 구현합니다.
- 소스, 에셋, 테스트, 설정은 가능한 한 해당 게임 디렉토리 내부에 둡니다.
- 루트 설정 변경이 필요하면 변경 이유를 커밋 또는 문서에 명확히 남깁니다.

## 커밋 메시지

- 커밋 메시지는 `prefix: 한글 설명` 형식을 사용합니다.
- 예: `docs: 게임 독립 개발 가이드 추가`
