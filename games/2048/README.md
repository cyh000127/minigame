# 2048

같은 숫자 타일을 합쳐 2048 타일을 만드는 퍼즐 게임입니다.

## 기술 스택

- TypeScript
- Vite
- Vitest

## 실행 방법

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 개발 서버는 `http://127.0.0.1:5178`을 사용합니다.

## 스크립트

```bash
corepack pnpm run build
corepack pnpm run preview
corepack pnpm run test
```

## 조작

- `ArrowUp`: 위로 밀기
- `ArrowRight`: 오른쪽으로 밀기
- `ArrowDown`: 아래로 밀기
- `ArrowLeft`: 왼쪽으로 밀기
- `U`: 직전 이동 1회 되돌리기
- 터치 화면에서는 스와이프로 이동

## 게임 규칙

- 같은 숫자의 타일 두 개가 같은 방향으로 밀리면 하나로 합쳐집니다.
- 한 타일은 한 번의 이동에서 한 번만 합쳐집니다.
- 이동이 실제로 일어난 뒤 빈 칸에 `2` 또는 `4` 타일이 새로 생성됩니다.
- `2048` 타일을 만들면 승리하며, 이후 계속 진행할 수 있습니다.
- 더 이상 이동하거나 합칠 수 없으면 게임이 종료됩니다.
- 최고점은 브라우저 `localStorage`에 저장합니다.
- `Goal` 선택으로 1024, 2048, 4096 목표 타일을 고를 수 있습니다.
- 실제 이동이 일어난 직전 상태는 한 번만 되돌릴 수 있습니다.
- 진행 중인 보드와 되돌리기 상태는 브라우저 `localStorage`에 자동 저장되며, 다시 열면 이어서 플레이합니다.
- 복구된 세션은 저장 시각, 점수, 이동 횟수를 상단 배너로 안내하며, 배너에서 즉시 새 게임으로 전환할 수 있습니다.

## 구조

```text
src/
  game.ts       2048 보드 이동, 병합, 점수, 승패 판정 엔진
  game.test.ts  게임 엔진 단위 테스트
  main.ts       브라우저 진입점, 입력, 렌더링, 점수 저장
  styles.css    2048 화면 스타일
```

이 게임은 `games/2048` 내부 코드만 사용하며, 다른 게임 디렉토리의 코드를 참조하지 않습니다.
