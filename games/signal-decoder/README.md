# Signal Decoder

방향 신호 시퀀스를 앞에서부터 정확한 순서로 입력해 해독하는 반응형 스테이지 게임입니다.

## 실행

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 주소는 `http://127.0.0.1:5195`입니다.

## 조작

- 방향키: 현재 신호 입력
- `N`: 새 런 시작
- `P`: 일시정지 또는 재개
- 마우스 클릭: 방향 버튼 입력

## 규칙

- 큐 맨 앞의 방향을 정확히 입력해야 합니다.
- 시퀀스를 모두 끝내면 스테이지 클리어입니다.
- 스테이지를 클리어하면 더 긴 시퀀스로 즉시 리셋되고 점수가 누적됩니다.
- 오입력은 스트라이크와 시간 패널티를 줍니다.
- 스트라이크 3회 또는 시간초과 시 런이 종료되고 누적 점수가 기록됩니다.

## 점수

- 난이도, 스테이지, 남은 시간, 시퀀스 길이에 따라 점수를 얻습니다.
- 스트라이크가 많을수록 스테이지 보너스가 줄어듭니다.
- 개인 최고 점수는 브라우저 `localStorage`에만 저장됩니다.

## 허브 연동

- 게임 시작 시 `minigame:start` 메시지를 부모 창에 전송합니다.
- 점수 변경 시 `minigame:score` 메시지를 전송합니다.
- 실패 또는 허브 종료 명령 시 `minigame:game-over` 메시지를 전송합니다.
- 허브의 `start`, `pause`, `gameOver` 라이프사이클 명령을 처리합니다.
