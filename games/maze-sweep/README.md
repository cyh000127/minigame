# Maze Sweep

랜덤 미로에서 오브를 모두 수집한 뒤 출구로 탈출하는 스테이지형 격자 게임입니다.

## 실행

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 주소는 `http://127.0.0.1:5194`입니다.

## 조작

- 방향키: 플레이어 이동
- `N`: 새 런 시작
- `P`: 일시정지 또는 재개

## 규칙

- 노란 오브를 모두 먹기 전에는 출구를 사용할 수 없습니다.
- 벽은 통과할 수 없습니다.
- 각 라운드에는 제한 시간이 있습니다.
- 제한시간 안에 클리어하면 점수를 얻고 새 랜덤 미로로 즉시 넘어갑니다.
- 시간초과 시 실패 처리되고 그 시점의 누적 점수가 기록됩니다.

## 맵 생성

- 플레이어 시작점, 출구, 모든 오브가 같은 연결 영역에 들어가도록 생성합니다.
- 따라서 스폰된 미로는 항상 클리어 가능합니다.

## 점수

- 난이도, 라운드, 남은 시간, 오브 수에 따라 점수를 얻습니다.
- 이동 횟수가 많을수록 라운드 보너스가 줄어듭니다.
- 개인 최고 점수는 브라우저 `localStorage`에만 저장됩니다.

## 허브 연동

- 게임 시작 시 `minigame:start` 메시지를 부모 창에 전송합니다.
- 점수 변경 시 `minigame:score` 메시지를 전송합니다.
- 시간초과 또는 허브 종료 명령 시 `minigame:game-over` 메시지를 전송합니다.
- 허브의 `start`, `pause`, `gameOver` 라이프사이클 명령을 처리합니다.
