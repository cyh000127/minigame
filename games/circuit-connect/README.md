# Circuit Connect

전원 타일에서 목표 배터리까지 회로 타일을 회전해 모든 활성 회로에 전류를 흘리는 퍼즐 게임입니다.

## 실행

```bash
corepack pnpm install
corepack pnpm run dev
```

기본 주소는 `http://127.0.0.1:5192`입니다.

## 조작

- 마우스 클릭: 회로 타일 회전
- 방향키: 포커스 이동
- Space 또는 Enter: 포커스된 타일 회전
- `N`: 새 퍼즐
- `P`: 일시정지 또는 재개

## 규칙

- 왼쪽 위 `P` 타일이 전원입니다.
- 오른쪽 아래 `B` 타일이 목표 배터리입니다.
- 회로 타일은 클릭할 때마다 시계 방향으로 90도 회전합니다.
- 모든 활성 회로 타일과 배터리가 전원에 연결되면 클리어됩니다.
- 각 라운드에는 제한 시간이 있습니다.
- 제한 시간 안에 클리어하면 점수를 얻고 새 맵으로 즉시 리셋됩니다.
- 제한 시간이 끝나면 실패로 처리되고 그 시점의 누적 점수가 기록됩니다.

## 랜덤 생성

- 생성기는 먼저 전원에서 배터리까지 도달 가능한 정답 회로를 만듭니다.
- 이후 각 타일의 회전 상태만 무작위로 섞습니다.
- 따라서 랜덤 퍼즐이어도 클리어가 불가능한 보드는 생성하지 않습니다.

## 점수

- 라운드 클리어 시 난이도, 라운드, 남은 시간, 활성 타일 수에 따라 점수를 얻습니다.
- 회전 횟수가 많을수록 라운드 보너스가 줄어듭니다.
- 개인 최고 점수는 브라우저 `localStorage`에만 저장됩니다.

## 허브 연동

- 게임 시작 시 `minigame:start` 메시지를 부모 창에 전송합니다.
- 점수 변경 시 `minigame:score` 메시지를 전송합니다.
- 클리어 또는 허브 종료 명령 시 `minigame:game-over` 메시지를 전송합니다.
- 허브의 `start`, `pause`, `gameOver` 라이프사이클 명령을 처리합니다.
